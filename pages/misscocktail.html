Miss Cocktail

Created with Ruby on Rails, Sass, Bootstrap, deployed on Heroku

Check it out here

Description

This basic Ruby on Rails CRUD application with MVC patternn was designed(barely), implemented and shipped to production in 2 days. It was my first Rails project during Le Wagon coding bootcamp. The database has a simple one to many relationship. A user is able to create a cocktail, while uploading an image and adding ingredients, update the ingredients, visit each cocktail recipe, remove a cocktail and its ingredients/image. A basic reviews controller was also added, belonging to a cocktail. The application was deployed to Heroku

Features

  • IMAGE UPLOAD: The Cloudinary gem was added in order to enable image uploading.
  • REVIEWS: A reviews model was added with a view to create a new review belonging to a cocktail
